# PRD: GeekSalon 受講生作品集サイト

## 概要

受講生の過去作品をギャラリー形式で閲覧できるサイト。
受講生が自分で作品を投稿し、現役受講生やメンターが参考として閲覧する。

## 技術構成

| レイヤー | 技術 |
|---|---|
| フレームワーク | Next.js (App Router) |
| ホスティング | Vercel |
| データベース | Neon (PostgreSQL) |
| 画像ストレージ | Cloudflare R2 (S3互換) |
| UI | React + Tailwind CSS v4 + shadcn/ui |
| 認証 | Auth.js (NextAuth v5) + Google OAuth |

## コース

- Web
- Web Expert
- Game
- AI
- Portfolio

## 認証・権限

| 操作 | 認証 |
|---|---|
| 閲覧 | 不要（一般公開） |
| 投稿・編集 | Googleログイン必須 + 招待リンクで登録 |

### 招待リンク方式

- 管理者が各期ごとに招待リンクを発行
- Slack等で受講生に共有
- 受講生がリンクからGoogleログインしてアカウント登録
- 以降は普通にログインして投稿可能

## 画面構成

| ページ | パス | 概要 |
|---|---|---|
| トップ / 一覧 | `/` | 作品カード一覧。コース別フィルタ。新着順 |
| 作品詳細 | `/works/[id]` | 作品の全情報表示 |
| 投稿フォーム | `/works/new` | 画像アップロード + テキスト入力 |
| ログイン | `/login` | Google OAuth |

## 入力項目（投稿フォーム）

| フィールド | 型 | 必須 | 備考 |
|---|---|---|---|
| 作品名 | テキスト | ○ | |
| URL | URL | ○ | デプロイ済みのURL |
| スクリーンショット | 画像 | ○ | R2に保存 |
| 技術スタック | テキスト | ○ | React, Unity 等 |
| 説明文 | テキスト（長文） | ○ | 作品の概要 |
| 制作者名 | テキスト | ○ | |
| 制作期間 | テキスト | ○ | 例: 2ヶ月 |
| コース / 期 | セレクト | ○ | 例: Web Expert 25期 |
| こだわりポイント | テキスト（長文） | △ | 任意 |

## 表示・ソート

- 新着順（デフォルト）
- コース別フィルタリング

## データフロー

```
受講生がサイト内フォームから投稿
  → Cloudflare R2（画像）
  → Neon PostgreSQL（テキスト情報）
    → Next.js App Router (RSC) でデータ取得して表示
```
