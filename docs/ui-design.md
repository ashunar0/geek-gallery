# 画面設計: GeekSalon 受講生作品集サイト

OOUI（オブジェクト指向UIデザイン）に基づく設計。

## オブジェクト

| オブジェクト | 説明 |
|---|---|
| Work（作品） | メインオブジェクト。ギャラリーの主役 |
| User（ユーザー） | 投稿者。作品に紐づく |

## ビュー構造

```
Work
  コレクション: カード一覧（トップ）
  シングル:     作品詳細

User
  シングル:     ユーザーページ（その人の作品一覧）
```

## プロパティ表示

### Work

| プロパティ | コレクション（カード） | シングル（詳細） |
|---|---|---|
| スクリーンショット | ○ サムネイル | ○ 大きく表示 |
| 作品名 | ○ | ○ |
| 制作者名 | ○ | ○ → Userページへリンク |
| コース / 期 | ○ | ○ |
| 技術スタック | ○ タグ表示 | ○ |
| 説明文 | △ 省略表示 | ○ |
| URL | - | ○ 外部リンク |
| 制作期間 | - | ○ |
| こだわりポイント | - | ○ |

### User

| プロパティ | シングル |
|---|---|
| アバター | ○ |
| 名前 | ○ |
| 投稿した作品一覧 | ○ Work のコレクション |

## アクション

### Work

| アクション | 条件 |
|---|---|
| 投稿 | 認証済み |
| 編集 | 本人のみ |
| 削除 | 本人のみ |

## ナビゲーションマップ

```
トップ（Work コレクション）
  │  フィルタ: コース別
  │
  ├─→ Work シングル（詳細）
  │     ├─→ URL（外部リンク）
  │     ├─→ User シングル（制作者ページ）
  │     └─→ 編集（本人のみ）
  │
  └─→ User シングル
        └─→ その人の Work コレクション
              └─→ Work シングル

ヘッダー
  ├─ ログイン / ログアウト
  └─ 投稿する（認証済みのみ表示）
```

## ページ一覧

| ページ | パス | 認証 | 説明 |
|---|---|---|---|
| トップ / 一覧 | `/` | 不要 | 作品カード一覧。コース別フィルタ。新着順 |
| 作品詳細 | `/works/[id]` | 不要 | 作品の全情報表示 |
| ユーザーページ | `/users/[id]` | 不要 | ユーザー情報 + その人の作品一覧 |
| 投稿フォーム | `/works/new` | 要 | 画像アップロード + テキスト入力 |
| 編集フォーム | `/works/[id]/edit` | 要（本人のみ） | 既存作品の編集 |
| ログイン | `/login` | 不要 | Google OAuth |
| 新規登録 | `/register` | 不要 | リンク非公開。受講生にのみURL共有 |
